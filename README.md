# powerfulservices

## Install 
   ```
    git clone https://github.com/4rth4S/powerfulservices.git
    cd powerfulservices/
    pip3 install -r requirements.txt 
   ```
## Usage
   ```
   python3 powerfulservices.py scan domain
   ```
### Example:
   ```
   python3 powerfulservices.py scan google.com
   ```
